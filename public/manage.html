Quick Upload – No JSON File Needed

Select the lecture, paste your questions array, and upload.

Paste the raw questions array exactly as it appears in your static DPP file.

Mathematical expressions supported: Use LaTeX syntax
  • Inline: \(x^2 + y^2 = z^2\)
  • Display: \[\frac{a}{b} = c\]
  • Powers: x^{2}, e^{i\pi}
  • Fractions: \frac{p}{q}
  • Roots: \sqrt{x}, \sqrt[3]{y}
  • Greek letters: \alpha, \beta, \gamma, \theta

Leave blank to use the selected subject's name.

Example Questions Format:

[
    {
        "date": "2023",
        "text": "Q1. Seven people Z, Y, X, W, V, U and T are sitting...",
        "options": ["Two", "Three", "One", "Four", "None of these"],
        "ans": 3,
        "diff": "MEDIUM",
        "explanation": "Detailed solution here..."
    }
]
