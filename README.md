# My PW - Online Classes Platform

A full-stack web application for students to watch lectures, mark completion, and access Notes & DPP.

### Features
- Separate sections for Banking, Quant, Current Affairs, English, Reasoning
- YouTube video integration
- Notes & More (PDF + DPP) with clean pop-up
- Admin panel to add/update lectures
- Mark lectures as completed

### Tech Stack
- Backend: Node.js + Express + MongoDB
- Frontend: HTML, Tailwind CSS, Vanilla JS

### How to Run Locally
```bash
npm install
npm run dev
